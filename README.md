# Operations Case Study: Student Engagement & Progress Tracking
### Project by: Deep Tiwari

## 📌 Project Overview
This project focuses on managing and optimizing student operations for a Bootcamp. It involves tracking real-time student progress, identifying "At-Risk" students using data analytics, and implementing a systematic communication strategy to improve engagement.

## 🚀 Key Features & Implementation

### 1. Data-Driven Student Tracker (Google Sheets/Excel)
- **Status Categorization:** Automated tracking of students into **Critical**, **At-Risk**, and **Active** categories based on weekly attendance and assessment submissions.
- **Attendance Monitoring:** Weekly tracking for sessions like Web Dev, Git, and JavaScript.
- **Automated Mail System:** Integrated logic to trigger "Confirmation Sent" or "Reminder" emails based on student activity.

### 2. Operational Dashboard (Visualization)
- **Bootcamp Weekly Report:** A visual dashboard showing session status (Conducted/Rescheduled).
- **Engagement Analytics:** Bar graphs and Donut charts to visualize the average batch percentage (currently 75-80%) and availability.
- **Insight Generation:** A summary section that highlights specific drops in engagement for proactive intervention.

### 3. Communication Strategy
- **Professional Reminders:** Drafted high-engagement student reminder templates for assessments.
- **Problem Solving:** A structured plan to tackle student absenteeism through Personal Outreach and the "Study Squads" (Buddy System) model.

## 🛠 Tools Used
- **Data Management:** Google Sheets / MS Excel (Advanced Formulas & Conditional Formatting)
- **Visualization:** Google Sheets Dashboards
- **Communication:** Gmail, Discord, WhatsApp
- **Strategy:** Root Cause Analysis (RCA) and Student Retention Frameworks

## 📈 Impact
- Successfully tracked 20+ students with a 100% completion rate for Coding Assessments.
- Identified the top 25% of students needing immediate operational support.
- Streamlined communication, reducing the manual effort of tracking student statuses.
